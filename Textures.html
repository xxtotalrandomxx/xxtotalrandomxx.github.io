Geometric Art, 2019

The Wrench is a piece created in 2-D concepts. It is a shadow figure of a wrench that only depicts the main detail of the object. This is meant to give the visual gesture that there is not only a blank area in the object, but also it is composed in many different styles.

I enjoyed this one very much because of how visually appealing it looks to me. I enjoy thinking about how it is shaped and how it uses black and white features to create a whole object.
