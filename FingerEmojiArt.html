Pinky Emoji

Adobe Illustrator, 2019

This Finger Emoji is just like those of the published hand gestures in emojis, but there is a finger that is not added to the selection and that is the pinky finger. The concept behind this was to create an emoji that would be used quite often if it were to be chosen into the emoticon pool.

I chose this because I felt like the emotes need each finger because it gives more options to use five different finger emojis than to just use the couple that are accepted.
